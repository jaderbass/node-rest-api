# node-rest-api

Beispiel-Projekt einer einfachen REST-API

## Hinweise

Ich habe in das Projekt noch eine Möglichkeit eingebaut Erfolgsmeldungen auszugeben. Dazu mussten noch die Module `express-session` und `connect-flash` installiert werden. Die Änderungen sind in der `app.js`, in `router.js` und im User-Dashboard (`all-users.ejs`) eingebaut und entsprechend gekennzeichnet. Wenn Ihr im VS Code die Erweiterung "Better Comments" installiert habt, seht ihr die Hinweise in blau.

Zum Installieren führt Ihr folgendes Kommando aus:

```bash
npm i express-session connect-flash
```

oder Ihr aktualisiert Euer lokales Repo über Github und updated es.
